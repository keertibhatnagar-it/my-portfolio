# My Portfolio Website

Welcome to my personal portfolio website! This project highlights my skills, projects, and experience as a frontend developer. Built with **Next.js**, it delivers a fast, responsive, and modern user experience.

---

## Features

* **Next.js Framework** for high performance and SEO
* **Tailwind CSS 4.0** for modern, clean, responsive UI
* **Fully Responsive** across all devices
* **Project Showcase Section** to display featured work
* **Dark Mode Support**
* **SEO Optimized**
* **Contact Form** to reach me easily

---

## Technologies Used

* **Framework:** Next.js
* **Styling:** Tailwind CSS 4.0
* **Icons:** Heroicons / Lucide
* **Deployment:** Vercel

---

## Getting Started

Follow these steps to run the project locally:

### Clone the Repository

```bash
git clone https://github.com/keertibhatnagar-it/my-portfolio.git
cd my-portfolio
```

### Install Dependencies

```bash
npm install
# or
yarn install
# or
pnpm install
```

### Start the Development Server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Now open **[http://localhost:3000](http://localhost:3000)** in your browser.
